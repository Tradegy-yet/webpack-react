# 依赖包说明
## package.json 
### dependencies：
* koa：
   * 说明：koa是Express的下一代基于Node.js的web框架，目前有1.x和2.0两个版本。
   * 版本： 2.0.0
* koa-router
   * 说明：负责处理URL映射。
   * 版本： 7.0.0
* koa-bodyparser
   * 说明：post请求url时，node.js和koa对象都不提供解析request body的功能，使用此中间件解析post参数。
   * 版本： 7.0.0
* koa-favicon
   * 说明：设置浏览器favicon.ico。
* koa-static
   * 说明：响应静态文件。
* react
   * 说明：react框架。
* webpack
   * 说明：模块打包器。
### devDependencies
* babel:
   * 说明：Babel是一个广泛使用的转码器，可以将ES6代码转为ES5代码，从而在现有环境执行。
